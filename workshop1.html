CMP9142 Workshop 1- Setup and Warm-up Activities

 

 

Introduction

This workshop is the first of many created to help you pick up the basics of cloud design and development, then progress through to more cloud infrastructure design and development. These workshops are designed to help you understand fundamental cloud concepts and as such it is vital you complete ALL of them. Therefore please take the time to go through all of the workshops. You can also continue working on the workshop tasks from home as the software requirements are basic (Internet browser). You are expected to go beyond the material and tasks presented in the workshops by attempting more advanced tasks and materials, some homeworks will be provided as well.

The primary goal of this workshop is to ensure you have your  Google Cloud hand-on labs and account setup, as well as familiarising yourself with the platform tools.

Google Cloud Skills Boost Platform

Cloud Skills Boost Platform (formerley known as Qwiklabs) is a learning platform that has deep integration with the Google Cloud Platform, essentially it provides a sandboxed environment for you to complete cloud labs through a browser. As a student on this module you have been given access to carry out many of the mandatory workshop tasks and tutorials as part of this module. Each lab exercise carries a 'credit' weighting, credits are required to do each lab task and these vary from lab to lab. The module coordinator has been allocated enough credits for all students on the module to complete all labs that are presented in each workshop.

When you carry out ANY lab exercise, please always remember that you must use the temporary GCP login details that are listed to the left of each lab - DO NOT login to the Google platform using your own GCP/gmail account details.
